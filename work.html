My Projects

Leap Motion Project

Computer Science degree dissertation project. I created an educational application using an infrared Leap Motion device. The system sucessfully detects and recognises sign language gestures by finding the Euclidean distance between each of the bone co-ordinates. Gestures are then stored in a database for the system to incrementally improve the accuracy for each gesture.
